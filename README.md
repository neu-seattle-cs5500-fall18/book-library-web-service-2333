# book-library-web-service-2333

- Back End

    - Python 2.7 flask server
    - Run `python2 app.py` to start
    
- Front End

    - React.js
    - To start:
        ```
        cd book-library
        npm install
        npm start
        ```
       
- Features so far

    - register
    - login
    - view all books
    - rent a book
    - return a book
